# FIT Web — Public-safe Overview

**FIT Web** is FIT's website solution family—covering the design, build, and automation infrastructure for client-facing websites. It combines a conversion-optimized web presence with clean lead capture, CRM integration, and automated intake workflows.

> Full tier details (Lite / Pro / Premium scope, entitlements, and support coverage) are available to clients and partners through your FIT contact.

---

## Who It Is For

Businesses and agencies that want a website which doesn't just look good—but actively captures leads, routes them cleanly, and connects into automation workflows without manual effort.

---

## What It Includes

FIT Web covers:

- Clean, fast website design built on a stable, maintainable stack
- Structured lead capture: contact forms with CRM routing and owner notification
- Intake automation: routing logic based on request type, source, or intent
- Optional AI chat integration (connects to FIT's knowledge system infrastructure)
- CRM hygiene: field mapping, deduplication, and pipeline entry
- Operator guidance: what to monitor, what failure looks like, how to request changes

Scope, included features, support coverage, and change entitlements are defined by tier (Lite, Pro, Premium).

---

## What It Is Not

- A generic website template or no-code website builder
- A standalone design project without automation infrastructure
- A substitute for a full marketing or brand agency
