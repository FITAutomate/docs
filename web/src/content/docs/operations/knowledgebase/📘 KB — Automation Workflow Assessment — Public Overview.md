# Automation Workflow Assessment — Public Overview

The **Automation Workflow Assessment** is a structured engagement that maps your key workflows, identifies the best automation opportunities, and delivers a prioritized roadmap—so you know exactly what to build first and why.

If you've already done an AI Readiness Audit, the Workflow Assessment goes deeper: it covers multiple workflows in detail, estimates time savings, and produces a phased 30/60/90 plan you can execute.

---

## Who It Is For

A good fit when you:

- Have multiple workflows you want to improve but aren't sure where to start
- Work across more than one system (email + CRM + spreadsheets + accounting, etc.)
- Want a practical plan before committing to a full build
- Need automation done safely, with clear approvals and guardrails

---

## What It Includes

FIT leads the process end-to-end. You don't need documented processes—we extract the workflow by asking the right questions.

By the end, you receive:

- A **ranked list of automation opportunities** with expected outcomes, complexity, risk, and guardrails needed
- **Estimated time savings** for the highest-value workflows (illustrative; varies by volume)
- **Workflow notes and assumptions** — what we observed and what needs confirming before build
- **Suggested implementation order** — quick wins first, then deeper workflows
- A **30/60/90 day roadmap** — a phased plan that scales responsibly

---

## What You Provide (Low Effort)

- A list of daily/weekly tasks (rough is fine)
- The tools involved
- A few examples (screenshots, forms, emails — sanitized is fine)
- Known pain points and priorities

---

## What It Is Not

- A build or implementation project (that comes after)
- A guarantee of specific ROI outcomes
- A technical architecture document
