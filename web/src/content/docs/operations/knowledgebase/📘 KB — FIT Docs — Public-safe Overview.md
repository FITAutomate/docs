# FIT Docs — Public-safe Overview

**FIT Docs** is FIT's document management and knowledge base solution family—covering the design, structure, and governance of client-facing and internal document systems. It gives your team a clean, maintained KB corpus that stays accurate over time.

> Full tier details (Lite / Pro / Premium scope, entitlements, and support coverage) are available to clients and partners through your FIT contact.

---

## Who It Is For

Organizations that need their documentation to be structured, accurate, and usable—by humans and by AI systems. If you have SOPs, FAQs, policies, or onboarding materials that are scattered, outdated, or inconsistently formatted, FIT Docs brings order to that corpus.

---

## What It Includes

FIT Docs covers:

- KB library setup: file naming conventions, required metadata fields, folder structure
- Document authoring and migration to a consistent standard format
- PUBLIC-facing and internal visibility separation—documents intended for internal use are never exposed to public channels
- Review cycle tracking: owner assignment, last-updated monitoring, and scheduled review reminders
- Change request workflow: submitted → reviewed → approved → published
- Pre-publish validation: automated checks for banned terms, routing compliance, and metadata completeness
- AI retrieval readiness (at higher tiers): documents prepared and structured for use in FIT Knows AI assistants

Scope, included document volume, governance depth, and support coverage are defined by tier (Lite, Pro, Premium).

---

## What It Is Not

- A generic CMS or content platform
- A one-time document clean-up with no ongoing governance
- A publishing platform (FIT Docs governs the content; hosting is separate)
