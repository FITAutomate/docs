# Solutions

Use subfolders under `marketing/solutions/` for **solution packs**:

| Folder | Purpose |
| --- | --- |
| `agencies/` | Industry / segment solution pages |
| `fit-web/` | FIT Web offering |
| `fit-docs/` | FIT Docs offering |
| `fit-knows/` | FIT Knows offering |

On **Windows**, `Solutions` and `solutions` resolve to the **same path**. Use **lowercase** `solutions` only, and avoid nested `Solutions/Solutions/...` folders.

If content was lost during a bad rename or cleanup, restore from Git, backup, or **Cursor Local History** on the missing `.md` files.
